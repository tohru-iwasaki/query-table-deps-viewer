# query-table-deps-viewer
SQL のクエリを textarea に突っ込んで「create graph」を押すと、クエリ中のテーブルやサブクエリの依存関係をグラフとして表示してくれるツール。

# ローカルサーバー起動方法

以下のコマンドで事前に環境構築を行う。

```shell
yarn install
```

以下のコマンドでローカルサーバーが起動する。

```shell
yarn start
```
